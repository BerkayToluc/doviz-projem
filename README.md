# 📈 MBT Finance Terminal

A real-time financial tracking dashboard built with Python, Flask, and Supabase. Monitor live exchange rates, gold prices, and crypto assets in one centralized portfolio.

## 🚀 Production Deployment
Check out the live application here: [https://berkay-exchange-app.onrender.com/]

## ✨ Features
- **Live Market Data:** Real-time updates from Binance, Truncgil, and CNBC.
- **Portfolio Management:** Add your assets and dynamically track your total value.
- **Visual Analytics:** Interactive distribution charts powered by Chart.js.
- **Secure Auth:** User registration and session management via Supabase.
- **Shared JSON Cache:** High-performance data synchronization across server workers to prevent data conflicts.

## 🛠️ Tech Stack
- **Backend:** Python, Flask
- **Database & Auth:** Supabase (PostgreSQL)
- **Frontend:** HTML, CSS, JavaScript, Bootstrap 5, Chart.js
- **Data Scraping:** BeautifulSoup4, Requests
